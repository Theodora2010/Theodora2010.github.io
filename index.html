CNN Classification ECG

This project can classify atrial fibrilation and normal electrocardiogram

Within this project, an algorithm was developed based on the time-frequency representation of electrocardiogram (ECG) signals and the Convolutional Neuronal Network (CNN) architectural model for the automatic classification of atrial fibrilation (AF).

The presented algorithm transforms the 1D ECG signal into 2D models of frequency densities, allowing the latest techniques of the CNN-based machine learning method to automatically classify AF. It analyzes the temporal frequency characteristics of atrial and ventricular activities, allowing the detection of FA using a very short period (a beat) of the ECG.

Data Acquisition and Processing

I recently created an innovative prototype in medicine using IoT technology and Matlab programming with which people with heart problems or sports lovers can monitor their cardiac activities wearing a T-shirt with hardware components that transmit data taken from circuits via Bluetooth. Given that this concept does not yet exist on the market, I have composed a scientific article sent to the BIOCAS 2021 international conference to be held in Berlin.

Realization of a real-time ECG signal acquisition algorithm using the Matlab program.

Implementation of an algorithm in order to eliminate the frequent noises in the acquisition of the signal corresponding to the cardiac activity with the help of a filter passes above 0.5 Hz, the Wavelet transform and a moving mean filter.

Creating an algorithm for detecting characteristic R waves by using an adaptive threshold for the calculation of the vital parameter (heart rate).

Design of a purchased signal processing visualization application with an interface designed for ease of use by the user that contains a data saving function.
